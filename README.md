# hscli

基于TypeScript+Rollup构建的一个cli工具

# 安装
```
npm install -g hscli
# 或者
yarn global add hscli
```
你可以用这个命令来检查其版本是否正确：
```
hscli --version
```

# 升级
如需升级全局的 hscli 包，请运行：
```
npm update -g hscli
# 或者
yarn global upgrade --latest hscli
```

# rollup知识点总结
1.
rollup 提供了三种选项
  cjs: node.js 环境
  iife: 浏览器环境
  umd: 兼容环境, 同时支持 node.js 和浏览器
